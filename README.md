# .emacs.d

@takeokunn's emacs configuration.

## Setup

```
$ make install
$ make update
$ make link
```

## Dependency

### Libraries

```
* the silver searcher
* slime-helper
* sbcl
* npm
* composer
```

### Language Server Protocol

Should install `language server`.

[emacs-lsp/lsp-mode#supported-languages](https://github.com/emacs-lsp/lsp-mode#supported-languages)

* HTML
* CSS
* Golang
* Javascript
* Ruby
* PHP
